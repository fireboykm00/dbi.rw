# Digital Business Institute (DBI)

The **Digital Business Institute (DBI)** is a trade and investment support institution (TISI) and the capacity-building arm of the **Rwanda ICT Chamber**. Our mission is to accelerate the digitalization of Rwandan businesses, expand digital exports, and increase the global competitiveness of Rwandan ICT companies.

- **DBI Academy**: An internationally accredited training hub providing professional certifications and capacity-building programs.
- **Digital Professionals Network (DPN)**: A robust network connecting member companies with global experts in health, IT, finance, and project management.
- **Certified Directory**: A comprehensive registry of verified ICT companies and service providers in Rwanda.

### Prerequisites

- Node.js 18+
- pnpm (Recommended)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/fireboykm00/dbi.rw.git
   ```

2. Install dependencies:

   ```bash
   pnpm install
   ```

3. Start the development server:

   ```bash
   pnpm dev
   ```

4. Build for production:
   ```bash
   pnpm build
   ```
